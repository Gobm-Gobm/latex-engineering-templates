# LaTeX Engineering Templates

Reusable LaTeX templates for engineering documentation.

## Purpose
- Create consistent, professional engineering documents
- Separate layout, styling, and content
- Enable future automation (Python → LaTeX)
- Serve as a personal documentation standard for consulting work

## Structure
- `templates/` – reusable document skeletons
- `styles/` – fonts, colors, headings
- `examples/` – real documents using the templates
- `docs/` – design decisions and notes
